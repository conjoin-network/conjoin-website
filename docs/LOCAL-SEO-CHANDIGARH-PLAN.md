## Internal link map
- Home/Nav -> `/microsoft-365-chandigarh`, `/seqrite-chandigarh`.
- Local landing pages -> related plan pages + `/request-quote` + `/contact` + relevant knowledge pages.
- Knowledge article detail pages -> service links (Microsoft/Endpoint/Procurement routes) based on article tags.
- Sitemap and search index include all landing routes.

## CTA placement standard
- Hero: `Request Quote` + `Contact Sales`.
- Immediate action bar: `Call +91 9466663015`, `WhatsApp`, `Get Quote`.
- Bottom section: lead form + direct email fallback.

## Weekly execution cadence
1. Publish one new knowledge article mapped to one commercial page.
2. Refresh `last verified` date for two existing knowledge articles.
3. Add one internal link from a high-traffic page to a target local page.
4. Inspect Search Console coverage for newly added slugs.

## Ranking KPI checkpoints
- Impressions and CTR for top 10 mapped keywords.
- Indexed status for all target slugs.
- Lead conversion rate from local intent pages.
- Average position trend for Chandigarh and Tricity modifiers.
